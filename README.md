# 🌸 Online Flower Shopping Website

A beautiful and responsive **Online Flower Shopping Website** built using **HTML**, **CSS**, and **JavaScript**. This project allows users to browse a variety of flowers, add them to a cart, and complete a purchase through a smooth checkout experience.

---

## 📄 Pages

| Page | Description |
|------|-------------|
| `index.html` | Home page with featured flowers and promotions |
| `shop.html` | Browse all available flowers |
| `cart.html` | View and manage items in the cart |
| `checkout.html` | Enter delivery and payment details |
| `about.html` | About the shop and its story |
| `contact.html` | Contact form for customer queries |
| `thankyou.html` | Order confirmation page |

---

## ✨ Features

- 🌺 Attractive and responsive flower shop UI
- 🛒 Add to cart functionality
- 💳 Checkout flow with order confirmation
- 📱 Mobile-friendly responsive design
- 🎨 Custom styling with CSS
- ⚡ Interactive elements using JavaScript

---

## 🛠️ Tech Stack

- **HTML5** – Page structure and content
- **CSS3** – Styling, animations, and responsive layout
- **JavaScript** – Cart logic and interactivity

## 🚀 Getting Started

1. **Clone the repository**
```bash
   git clone https://github.com/DhivyaDharsini-2005/Online-Flower-Shopping-Website.git
```

2. **Open the project**
```bash
   cd Online-Flower-Shopping-Website
```

3. **Run it** — Simply open `index.html` in your browser. No server setup needed!
